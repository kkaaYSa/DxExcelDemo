## 🚀 Features

- **Minimalist UI:** All unnecessary Ribbon tabs (Home, Insert, Page Layout, etc.) are removed to provide a focused user experience.
- **Custom Context Menu:** Users can right-click any cell to see a "Değeri Göster" (Show Value) option.
- **Client-Side Integration:** Leverages DevExpress Client-Side API to handle spreadsheet actions without full page reloads.
- **Backend Architecture:** Built using ASP.NET MVC with a clean Partial View structure.
